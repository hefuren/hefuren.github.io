java.lang.OutOfMemoryжҖ»з»“еҲҶжһҗ

еҸ‘иЎЁдәҺ 2017-10-07 | еҲҶзұ»дәҺ зЁӢеәҸйҡҸи®° | йҳ…иҜ»ж¬Ўж•° |
еӯ—ж•°з»ҹи®Ў

гҖҖгҖҖеңЁи§ЈеҶіjavaеҶ…еӯҳжәўеҮәй—®йўҳд№ӢеүҚпјҢйңҖиҰҒеҜ№jvmпјҲjavaиҷҡжӢҹжңәпјүзҡ„еҶ…еӯҳз®ЎзҗҶжңүдёҖе®ҡзҡ„и®ӨиҜҶгҖӮjvmз®ЎзҗҶзҡ„еҶ…еӯҳеӨ§иҮҙеҢ…жӢ¬дёүз§ҚдёҚеҗҢзұ»еһӢзҡ„еҶ…еӯҳеҢәеҹҹпјҡ

  • Permanent Generation spaceпјҲж°ёд№…дҝқеӯҳеҢәеҹҹпјүпјҡж°ёд№…дҝқеӯҳеҢәеҹҹдё»иҰҒеӯҳж”ҫClassпјҲзұ»пјүе’ҢMetaзҡ„дҝЎжҒҜпјҢClassз¬¬дёҖж¬Ўиў«Loadзҡ„ж—¶еҖҷиў«ж”ҫе…ҘPermGen spaceеҢәеҹҹпјҢClassйңҖиҰҒеӯҳеӮЁзҡ„еҶ…е®№дё»иҰҒеҢ…жӢ¬ж–№жі•е’ҢйқҷжҖҒеұһжҖ§
  • Heap space(е ҶеҢәеҹҹ)пјҡе ҶеҢәеҹҹз”ЁжқҘеӯҳж”ҫClassзҡ„е®һдҫӢпјҲеҚіеҜ№иұЎпјүпјҢеҜ№иұЎйңҖиҰҒеӯҳеӮЁзҡ„еҶ…е®№дё»иҰҒжҳҜйқһйқҷжҖҒеұһжҖ§гҖӮжҜҸж¬Ўз”ЁnewеҲӣе»әдёҖдёӘеҜ№иұЎе®һдҫӢеҗҺпјҢеҜ№иұЎе®һдҫӢеӯҳеӮЁеңЁе ҶеҢәеҹҹдёӯпјҢиҝҷйғЁеҲҶз©әй—ҙд№ҹиў«jvmзҡ„еһғеңҫеӣһж”¶жңәеҲ¶з®ЎзҗҶгҖӮ
  • Java Stacks(Javaж ҲпјүпјҡJavaж Ҳи·ҹеӨ§еӨҡж•°зј–зЁӢиҜӯиЁҖеҢ…жӢ¬жұҮзј–иҜӯиЁҖзҡ„ж ҲеҠҹиғҪзӣёдјјпјҢдё»иҰҒеҹәжң¬зұ»еһӢеҸҳйҮҸд»ҘеҸҠж–№жі•зҡ„иҫ“е…Ҙиҫ“еҮәеҸӮж•°гҖӮJavaзЁӢеәҸзҡ„жҜҸдёӘзәҝзЁӢдёӯйғҪжңүдёҖдёӘзӢ¬з«Ӣзҡ„е Ҷж ҲгҖӮ

гҖҖгҖҖе®№жҳ“еҸ‘з”ҹеҶ…еӯҳжәўеҮәй—®йўҳзҡ„еҶ…еӯҳз©әй—ҙеҢ…жӢ¬пјҡPermanent Generation spaceе’ҢHeap spaceгҖӮ

з¬¬дёҖз§ҚOutOfMemoryErrorпјҡ PermGen space

гҖҖгҖҖеҸ‘з”ҹиҝҷз§Қй—®йўҳзҡ„еҺҹж„ҸжҳҜзЁӢеәҸдёӯдҪҝз”ЁдәҶеӨ§йҮҸзҡ„jarжҲ–classпјҢдҪҝjavaиҷҡжӢҹжңәиЈ…иҪҪзұ»зҡ„з©әй—ҙдёҚеӨҹпјҢдёҺPermanent Generation spaceжңүе…ігҖӮи§ЈеҶіиҝҷзұ»й—®йўҳжңүд»ҘдёӢдёӨз§ҚеҠһжі•пјҡ
1. еўһеҠ JavaиҷҡжӢҹжңәдёӯзҡ„XX:PermSizeе’ҢXX:MaxPermSizeеҸӮж•°зҡ„еӨ§е°ҸпјҢе…¶дёӯXX:PermSizeжҳҜеҲқе§Ӣж°ёд№…дҝқеӯҳеҢәеҹҹеӨ§е°ҸпјҢXX:MaxPermSizeжҳҜжңҖеӨ§ж°ёд№…дҝқеӯҳеҢәеҹҹеӨ§е°ҸгҖӮ
еҰӮй’ҲеҜ№tomcat6.0пјҢеңЁcatalina.sh жҲ–catalina.batж–Үд»¶дёӯдёҖзі»еҲ—зҺҜеўғеҸҳйҮҸеҗҚиҜҙжҳҺз»“жқҹеӨ„пјҲеӨ§зәҰеңЁ70иЎҢе·ҰеҸіпјү еўһеҠ дёҖиЎҢпјҡ

гҖҖгҖҖеҰӮжһңжҳҜwindowsжңҚеҠЎеҷЁиҝҳеҸҜд»ҘеңЁзі»з»ҹзҺҜеўғеҸҳйҮҸдёӯи®ҫзҪ®гҖӮж„ҹи§үз”ЁtomcatеҸ‘еёғsprint+struts+hibernateжһ¶жһ„зҡ„зЁӢеәҸж—¶еҫҲе®№жҳ“еҸ‘з”ҹиҝҷз§ҚеҶ…еӯҳжәўеҮәй”ҷиҜҜгҖӮдҪҝз”ЁдёҠиҝ°ж–№жі•пјҢжҲ‘жҲҗеҠҹи§ЈеҶідәҶйғЁзҪІsshйЎ№зӣ®зҡ„tomcatжңҚеҠЎеҷЁз»Ҹеёёе®•жңәзҡ„й—®йўҳгҖӮ

2. жё…зҗҶеә”з”ЁзЁӢеәҸдёӯweb-inf/libдёӢзҡ„jarгҖӮ
гҖҖгҖҖеҰӮжһңtomcatйғЁзҪІдәҶеӨҡдёӘеә”з”ЁпјҢеҫҲеӨҡеә”з”ЁйғҪдҪҝз”ЁдәҶзӣёеҗҢзҡ„jarпјҢеҸҜд»Ҙе°Ҷе…ұеҗҢзҡ„jarз§»еҲ°tomcatе…ұеҗҢзҡ„libдёӢпјҢеҮҸе°‘зұ»зҡ„йҮҚеӨҚеҠ иҪҪгҖӮиҝҷз§Қж–№жі•жҳҜзҪ‘дёҠйғЁеҲҶдәәжҺЁиҚҗзҡ„пјҢжҲ‘жІЎиҜ•иҝҮпјҢдҪҶж„ҹи§үеҮҸе°‘дёҚдәҶеӨӘеӨ§зҡ„з©әй—ҙпјҢжңҖйқ и°ұзҡ„иҝҳжҳҜз¬¬дёҖз§Қж–№жі•гҖӮ

з¬¬дәҢз§ҚOutOfMemoryErrorпјҡ Java heap space

гҖҖгҖҖеҸ‘з”ҹиҝҷз§Қй—®йўҳзҡ„еҺҹеӣ жҳҜjavaиҷҡжӢҹжңәеҲӣе»әзҡ„еҜ№иұЎеӨӘеӨҡпјҢеңЁиҝӣиЎҢеһғеңҫеӣһж”¶д№Ӣй—ҙпјҢиҷҡжӢҹжңәеҲҶй…Қзҡ„еҲ°е ҶеҶ…еӯҳз©әй—ҙе·Із»Ҹз”Ёж»ЎдәҶпјҢдёҺHeap spaceжңүе…ігҖӮи§ЈеҶіиҝҷзұ»й—®йўҳжңүдёӨз§ҚжҖқи·Ҝпјҡ
1. жЈҖжҹҘзЁӢеәҸпјҢзңӢжҳҜеҗҰжңүжӯ»еҫӘзҺҜжҲ–дёҚеҝ…иҰҒең°йҮҚеӨҚеҲӣе»әеӨ§йҮҸеҜ№иұЎгҖӮжүҫеҲ°еҺҹеӣ еҗҺпјҢдҝ®ж”№зЁӢеәҸе’Ңз®—жі•гҖӮ
2. еўһеҠ JavaиҷҡжӢҹжңәдёӯXmsпјҲеҲқе§Ӣе ҶеӨ§е°Ҹпјүе’ҢXmxпјҲжңҖеӨ§е ҶеӨ§е°ҸпјүеҸӮж•°зҡ„еӨ§е°ҸгҖӮеҰӮпјҡ

з¬¬дёүз§ҚOutOfMemoryErrorпјҡunable to create new native thread

гҖҖгҖҖиҝҷз§Қй”ҷиҜҜеңЁJavaзәҝзЁӢдёӘж•°еҫҲеӨҡзҡ„жғ…еҶөдёӢе®№жҳ“еҸ‘з”ҹпјҢжҲ‘жҡӮж—¶иҝҳжІЎйҒҮеҲ°иҝҮпјҢеҸ‘з”ҹеҺҹж„Ҹе’Ңи§ЈеҶіеҠһжі•еҸҜд»ҘеҸӮиҖғпјҡ

д»ҺJVMеұӮйқўеҺ»и§ЈеҶіпјҡеҮҸе°Ҹthread stackзҡ„еӨ§е°ҸпјҢJVMй»ҳи®Өthread stackзҡ„еӨ§е°Ҹдёә1024пјҢиҝҷж ·еҪ“зәҝзЁӢеӨҡж—¶еҜјиҮҙNative virtual memoryиў«иҖ—е°ҪпјҢе®һйҷ…дёҠеҪ“thread stackзҡ„еӨ§е°Ҹдёә128K жҲ– 256Kж—¶жҳҜи¶іеӨҹзҡ„пјҢжүҖд»ҘжҲ‘д»¬еҰӮжһңжҳҺзЎ®жҢҮе®ҡthread stackдёә128K жҲ– 256KеҚіеҸҜпјҢе…·дҪ“дҪҝз”Ё-XssпјҢдҫӢеҰӮеңЁJVMеҗҜеҠЁзҡ„JVM_OPTдёӯж·»еҠ еҰӮдёӢй…ҚзҪ®пјҡ

еҮҸе°ҸheapжҲ–permgenеҲқе§ӢеҲҶй…Қзҡ„еӨ§е°ҸпјҡеҰӮжһңJVMеҗҜеҠЁзҡ„JVM_OPTдёӯжңүеҰӮдёӢй…ҚзҪ®

жҲ‘д»¬еҸҜд»ҘеҲ йҷӨжҲ–еҮҸе°ҸеҲқе§ӢеҢ–жңҖе°ҸеҖјзҡ„й…ҚзҪ®пјҢеҰӮдёӢ

еҚҮзә§JVMеҲ°жңҖж–°зҡ„зүҲжң¬пјҡжңҖж–°зүҲжң¬зҡ„JVMдёҖиҲ¬еңЁеҶ…еӯҳдјҳеҢ–ж–№йқўеҒҡзҡ„жӣҙеҘҪпјҢеҚҮзә§JVMеҲ°жңҖж–°зҡ„зүҲжң¬еҸҜиғҪдјҡзј“и§ЈжөӢй—®йўҳгҖӮ

д»Һж“ҚдҪңзі»з»ҹеұӮйқўеҺ»и§ЈеҶіпјҡдҪҝз”Ё64дҪҚж“ҚдҪңзі»з»ҹпјҢеҰӮжһңдҪҝз”Ё32дҪҚж“ҚдҪңзі»з»ҹйҒҮеҲ°unable to create new native threadпјҢе»әи®®дҪҝз”Ё64дҪҚж“ҚдҪңзі»з»ҹгҖӮ

еўһеӨ§OSеҜ№зәҝзЁӢзҡ„йҷҗеҲ¶пјҡеңЁLinuxж“ҚдҪңзі»з»ҹи®ҫе®ҡnofileе’ҢnprocпјҢе…·дҪ“зј–иҫ‘/etc/security/limits.confж·»еҠ еҰӮдёӢпјҡ

еҰӮжһңдҪҝз”ЁRed Hat Enterprise Linux 6пјҢзј–иҫ‘/etc/security/limits.d/90-nproc.confпјҢж·»еҠ еҰӮдёӢй…ҚзҪ®пјҡ
